# datasets/

This directory holds raw and processed data. All data files are gitignored — only this README is tracked.

## Raw Data

Raw sources are hosted on Hugging Face: [farefaine/tigrinya-pretraining](https://huggingface.co/datasets/farefaine/tigrinya-pretraining)

Download to `datasets/raw/` before running the pipeline:

```bash
hf download farefaine/tigrinya-pretraining --repo-type=dataset --local-dir datasets/raw/
```

## Structure

```
datasets/
├── raw/          # Raw source files (downloaded from HF, gitignored)
│   ├── en-ti-bible/
│   ├── tigqa/
│   ├── tigrinya_sentences.txt
│   ├── train.txt
│   └── validation.txt
└── training/     # Processed splits (generated by pipeline, gitignored)
    ├── train.txt
    ├── validation.txt
    └── test.txt
```

## Generating the Processed Splits

After downloading raw data, run from the project root:

```bash
python -m data.build_dataset
```

See `data/README.md` for full pipeline documentation.
