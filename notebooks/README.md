# notebooks/

Exploratory notebooks for analysis, experiments, and project discovery.

## Data

| Notebook | Purpose |
|---|---|
| `00_data_discovery.ipynb` | Explores raw dataset size, duplicates, vocabulary frequency, and wordclouds before preprocessing/training. |

Run notebooks from the project root, or ensure the project root is available on `sys.path`.

The data discovery notebook expects raw files under `datasets/raw/` and uses helpers from `data/dataset_stats.py`.
